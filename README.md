# AdmobAds
Easy Admob Ads Library

> Step 1. Add the JitPack repository to your build file
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  > Step 2. Add the dependency
  ```
  	dependencies {
	        implementation 'com.github.AtikulSoftware:AdmobAds:1.0.0'
	}
  ```
